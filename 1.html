Prim's Algorithm Visualizer

Interactive visualization of Prim's Algorithm for finding the Minimum Spanning Tree (MST) in a weighted undirected graph. Watch as the algorithm builds the MST step by step, always selecting the minimum weight edge that connects a node in the MST to a node outside it.

Algorithm Information

Time & Space Complexity

Time Complexity

O(E log V)

Using adjacency list and priority queue

Space Complexity

O(V + E)

For storing graph and priority queue

Key Concepts

  • Greedy algorithm - makes locally optimal choice at each step
  • Finds Minimum Spanning Tree (MST) for weighted undirected graphs
  • MST connects all vertices with minimum total edge weight
  • Works by growing the MST one vertex at a time
  • Always selects the minimum weight edge connecting MST to non-MST vertices
